# **App Name**: CodeClarity

## Core Features:

- Automated Code Review: Uses AI to automatically review code submissions for common errors, style inconsistencies, and potential bugs.
- Code Style Enforcement: Checks code against a pre-defined style guide to ensure consistency across the project.
- Security Vulnerability Detection: AI powered tool that detects potential security vulnerabilities such as SQL injection, cross-site scripting (XSS), and other common exploits.
- Code Quality Scoring: Provides a quality score based on factors such as complexity, maintainability, and test coverage.
- Integration with Git: Seamlessly integrates with Git repositories to review code as part of the pull request process.
- Personalized Recommendations: Suggests personalized improvements to the code based on best practices and the developer's skill level.

## Style Guidelines:

- Primary color: Midnight blue (#2C3E50) for a professional and trustworthy feel.
- Background color: Light gray (#F0F4F8), same hue as primary, but desaturated for neutrality.
- Accent color: Electric purple (#8E44AD), contrasting brightness and saturation, sitting to the 'left' on the color wheel, creating emphasis for alerts and important actions.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective look. Note: currently only Google Fonts are supported.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear and concise icons to represent different types of code issues and suggestions.
- Maintain a clean and organized layout to present code review findings in an easy-to-understand manner.
